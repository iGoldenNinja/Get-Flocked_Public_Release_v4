Welcome to the Get-Flocked Signal Deck discussions space.

This project is a public, clean-slate M5Stack FIRE firmware release derived from FlockSquawk and heavily modified for passive RF awareness, field logging, debug tools, tone audio, and optional GPS/geofence support.

Use Discussions for:

* Setup questions for Arduino IDE, M5Stack FIRE, SD cards, audio tones, logging, and GPS modules.
* Field-test notes, accuracy reports, and repeatable observations.
* Ideas for improving detection logic, debug tools, menus, documentation, or GPS/geofence workflows.
* Sharing clean, non-personal test data that helps improve the public firmware.
* Helping new users get started.

Please remember:

* GPS/geofencing requires a compatible UART GPS module installed in the M5Stack FIRE module slot or connected to the configured GPS port and a valid satellite fix.
* RF detection can still work without GPS.
* Do not post private coordinates, personal logs, private addresses, or sensitive local watch-list data.
* Keep exact local MAC watch-list entries private unless you are certain they are safe and appropriate to share.
* Be respectful, curious, and open-minded. This is a community we build together.

To get started, comment below with:

* What hardware you are using.
* Which firmware version you are testing.
* Whether you are using GPS/geofencing or RF-only mode.
* What you want to improve, test, or learn.
